# Status

1. **Read first** — `AGENTS.md`, `docs/Current-State.md`, `docs/README.md`, and `docs/generated/` as needed for scope.

2. **Optional dispatch** — For codebase-wide or ambiguous questions:
   - **`explore`** — systematic coverage (set thoroughness: quick, medium, very thorough).
   - **`technical-writer`** — structured synthesis from docs when the user wants a polished narrative.

3. **Synthesize** — Executive summary, key findings with evidence (repo **file paths** only), actionable recommendations.
